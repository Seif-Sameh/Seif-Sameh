## `> cat ./projects.log`

<br/>

### 🔗 Kaggle MCP Server
> **Python · Model Context Protocol · Kaggle API**

Built an MCP server exposing **39 Kaggle tools** — competitions, datasets, kernels, models. Search, filter, version control, all wired up cleanly. **Merged into the official MCP community repository.**

<br/>

### 📄 Smart Contract Assistant
> **LangChain · RAG · FastAPI · ChromaDB / FAISS · SentenceTransformers**

End-to-end RAG system for contract analysis. Clause-level PDF/DOCX ingestion → semantic chunking → vector retrieval → Groq-powered answers. Multi-turn conversations, summarization, and safety guardrails out of the box.

<br/>

### 🤖 RoboChat
> **LangGraph · OpenAI · ChromaDB · TavilyAPI · SerpAPI · FastAPI**

Agentic RAG system for robotics datasheets. Real-time web search via Tavily + geolocation-based component lookup via SerpAPI. Built for engineers who need answers fast.

<br/>

### 🏃 Human Activity Recognition
> **TensorFlow · LSTMs · Scikit-learn**

LSTM classifier on WISDM v1.1 accelerometer data. **95% accuracy · 96% F1-score.** Clean pipeline from raw sensor data to deployment-ready model.

<br/>

### 🧾 Invoice Data Extraction
> **PyTorch · LayoutLMv3 · OpenCV · SROIE Dataset**

Fine-tuned LayoutLMv3 on scanned invoice data. **95.8% F1-score** on fields like date, total, and invoice number. Outputs structured JSON for downstream automation.
